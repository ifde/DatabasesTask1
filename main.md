# ДЗ №1 Базы Данных

## Роли пользователей
1. Клиент - выбирает блюда, создает заказ, оплачивает заказ, принимает доставку
2. Ресторан - готовит и собирает заказ, передает курьеру
3. Курьер - доставляет заказ
